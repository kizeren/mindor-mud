




This is a universal MOB table that can be implemented into any ROM 2.4 or ROT v1.2-1.5
MUD with up to 250 mortal levels.

The tables that are included are:

        1. MOB HITDICE
        2. MOB DAMDICE
        3. MOB ARMOR CLASS
        4. WEAPON DICE VALUES
        5. OBJECT BALANCING

 Hopefully, these values can be used in order to create a mainframe for
 area builders that enable easy crossover of areas from Mud to Mud :)

   Values can be written in the following HITDICE formula:

                 To Calculate Bonus
                                                        
(Hit Points per level) * (Mob Level / 4) =( BONUS TO HITDICE)

                 To Configure Hitdice:
                                                         
(Mob Level / 2) * (Hit Points per level * 1.5) + (BONUS TO HITDICE)
---------------------------------------------------------------------------------------------------------------------------------------
                                                 
i.e  Level 130 MOB with High End Hit Points
                                                                     
Bonus to Hitdice:                                                         
                                                       
(130 hp/level) * (130-Mob Level / 4) 
                                                                 
(130 hp/level) * (32.5)
                                                                  
= 4225 Bonus to Hitdice 

                                                                    
 Calculate Hitdice:
                                
(Mob Level / 2) * (Hit Points per Level * 1.5) + (BONUS TO HITDICE)                                                   
                                                              
                  (130/2) * (130 * 1.5) + 4225
                                                                   
                      (65) * (195) + 4225

                                               
         Hitdice for a Level 130 Mob would be 65d195+4225
------------------------------------------------------------------------------------------------------------------------------------------------
                           
NOTE: You can simplify HITDICE total by dividing the number of dice 
                           
rolled so that it is less than 20 and then multiplying the ratio to  
                           
the dice size. 
                                
i.e  Configured Hitdice for Level 130 Mob 65d195+4225 can be simplified
                                
by dividing the number of dice thrown by a number ranging from 10-15.


              Multiply the size of the dice by RATIO
                        195 * 5 = 975

        Simplified HITDICE for a Level 130 Mob = 12d975+4225

     *******************************************************************   
                               OBJECT BALANCING
     *******************************************************************
     A. SCROLLS AND POTIONS
     1. Object level should be no less than the minumum level required         for any class to cast the spell normally.
     2. The maximum level of the spell should not exceed the level of                                 the object by more than 10 or 40%, whichever is GREATER.
     3. The value of the item should reflect the power of the spell.

     B. WANDS AND STAVES
     1. Use same level restrictions as scrolls and potions.
     2. The number of charges should be inversely proportional to the  
        power of the spell.  
       (Basically, a level 40 staff will have more charges 
        of a level 30 spell than it would of a level 40 spell.)
     3. Usage of the more powerful/popular spells (heal,sanc,invis, etc)
        should max out between 1-5 charges.

     C. ARMOR
     1. Value for each AC type should not exceed 1/5 of the item level.

        i.e.  A level 50 breastplate armor should not exceed 10.
              50 / 5 = 10  ........ROUND DOWN

     2. Value for each AC type exotic should not exceed 1/8 of item         level.
   
        i.e   A level 50 breastplate exotic should not exceed 6.


     STAT: A level 50 breastplate AC should not exceed  10 10 10 6

     3. It is a good practice to vary the different armor class ratings
        at or below the maximum to suit armor type.

        i.e  A level 50 breastplate may offer greater armor class vs.   
             bash than it would be vs. pierce and slash, so the AC for 
             bash could reflect higher AC in that stat without exceeding 
             the maximum allowable AC, while the AC for pierce and slash 
             could be made lower than maximum AC allowed at object              level.

 The formula used for configuring weapon averages using dice xdy is:

                    (x * y) + x / 2 = weapon average

                    i.e. a level 100 weapon uses dice 6d17

                    (6 * 17) + 6 / 2 = 54 avg

     Average damage should not exceed (0.5 * weapon level) + 4

     Alternative:
     Maximum should be an average 5 to start plus 5 for every 10 levels.

                   i.e  level 100 weapon
             5 + 50 (start average + (100 / 10 = 10 * 5) + 4 = 59
                      level 100 = 59 avg
